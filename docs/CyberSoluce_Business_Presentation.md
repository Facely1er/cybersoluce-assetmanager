## Slide 6: Feature 2 - Software Identification & SBOM Analysis

### TechnoSoluce Integration

**SBOM Upload & Analysis**
- **Supported Formats**: SPDX (JSON/tag-value) and CycloneDX (JSON/XML)
- **Direct Integration**: Upload SBOMs directly to TechnoSoluce from CyberSoluce
- **Asset Linking**: Link SBOMs to specific assets in your inventory
- **Automatic Processing**:
  - Component metadata extraction
  - Dependency graph construction
  - Vulnerability correlation
  - Software supply chain mapping

**What TechnoSoluce Provides**
- **Component Analysis**: Detailed breakdown of software components and versions
- **Dependency Mapping**: Visualize software dependencies and relationships
- **Vulnerability Intelligence**: Correlate components with known vulnerabilities
- **Supply Chain Visibility**: Track software origins and dependencies

**Workflow**
1. Identify software assets in CyberSoluce
2. Upload SBOM files (one per asset or aggregated)
3. Link SBOMs to corresponding assets
4. TechnoSoluce analyzes and returns component intelligence
5. Signals flow back to CyberSoluce for visibility

**Key Benefits**
- ✅ **Complete Software Visibility**: Know exactly what's in your applications
- ✅ **Risk Identification**: Identify vulnerable components automatically
- ✅ **Compliance Support**: Document software composition for compliance requirements
- ✅ **Supply Chain Security**: Track third-party and open-source dependencies
